# 🚀 Smart Accounting System - Gold Standard Edition

نظام محاسبي متطور مبني بأحدث المعايير البرمجية لضمان الاستقرار، الأداء العالي، وقابلية التوسع.

## 🏗️ هندسة النظام (Architecture)
تم تصميم هذا المشروع وفقاً لـ **Gold Standard Architecture**، والتي تعتمد على:
- **Clean Architecture:** فصل تام بين منطق الأعمال (Business Logic) وواجهات العرض.
- **DTOs & Explicit Projections:** ضمان كفاءة استعلامات قاعدة البيانات ومنع تسريب البيانات.
- **Service Layer Pattern:** مركزية العمليات الحسابية والمالية لضمان دقتها.

## 🌟 المميزات الرئيسية
- **محرك فواتير متطور:** دعم كامل لدورة حياة الفاتورة (مسودة، مؤكدة، مدفوعة).
- **إدارة المخزون:** تتبع تلقائي لحركات المخزون مع نظام حماية من "الكميات السالبة".
- **ذكاء اصطناعي مدمج:** تحليلات تنبؤية للمبيعات والمخزون.
- **دعم ZATCA:** جاهز للتكامل مع متطلبات الفوترة الإلكترونية.

## 🛠️ التقنيات المستخدمة
- **Backend:** .NET 8 Web API
- **Frontend:** WinForms (Production Ready) & Blazor (Agent)
- **Database:** SQLite (Entity Framework Core)
- **AI Engine:** Python/Ollama Integration

## 🚀 كيفية التشغيل
1. قم بتحميل المستودع: `git clone https://github.com/moh9867/SmartAccounting_Full_Project.git`
2. افتح الحل (Solution) باستخدام Visual Studio 2022.
3. تأكد من إعداد قاعدة البيانات عبر تشغيل مشروع `DbSeeder`.
4. ابدأ مشروع `SmartAccountingSystem.Api` و `SmartAccountingSystem.Production`.
